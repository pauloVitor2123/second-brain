## Obsidian Web Clipper como Chrome Extension

Extensão instalável no Chrome que permite capturar web diretamente.

## Instalação

1. Ir em obsidian.md
2. Navegar até seção web clipper
3. Clicar "Add to Chrome"
4. Apareça ícone em barra do Chrome

## Uso

Clicar ícone → captura página → "Add to Obsidian" → salva em vault configurado

## Vantagens

- Uma ação (um clique) para capturar
- Não precisa deixar navegador
- Automático com metadados
