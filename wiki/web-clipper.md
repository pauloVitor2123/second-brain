## O que é

Extensão de navegador que permite capturar conteúdo web e salvá-lo em formato estruturado.

## Obsidian Web Clipper

Específicamente para Obsidian:
- Captura página inteira como markdown
- Extrai automaticamente transcrições de vídeos YouTube
- Salva metadados (title, URL, data, tags)
- Envia direto para pasta configurada no vault (`/raw`)
- Preenche front matter com propriedades do conteúdo

## Fluxo

1. Navegar na web (artigo, vídeo, tweet)
2. Clicar ícone Web Clipper
3. Revisar conteúdo capturado
4. "Add to Obsidian"
5. Arquivo salvo em `/raw` automaticamente

## Vantagem

Automatiza primeira etapa de ingestão — não precisa copiar/colar manualmente.

## Configuração

Em Web Clipper settings:
- Vault de destino
- Pasta padrão (recomendado: `/raw`)
- Template de propriedades
- Tags automáticas
