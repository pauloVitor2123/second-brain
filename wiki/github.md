## Uso no segundo cérebro

Backup remoto de todo o vault do Obsidian:
- Criar repositório privado com nome do projeto (ex: "second-brain")
- Conectar Codex via plugin GitHub
- Fazer commit automático após processar novos arquivos
- Toda mudança fica registrada + recuperável

## Fluxo

1. Obsidian salva localmente (markdown)
2. Codex processa
3. Automação faz push para repositório privado no GitHub
4. Histórico de versões preservado
