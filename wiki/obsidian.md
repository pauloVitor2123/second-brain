## O que é

Organizador de markdown gratuito e open-source que funciona como frontend para sistemas de gestão de conhecimento. Armazena notas em formato markdown organizado em pastas (vaults).

## Características principais

- Armazenamento local de arquivos markdown
- Visualização interativa (graph view) mostrando conexões entre notas
- Suporte a links internos com síntaxe `[[nome-da-nota]]`
- Web clipper integrado para capturar conteúdo da web

## Obsidian Web Clipper

Extensão do Chrome que:
- Captura páginas web completas e salva como markdown
- Extrai automaticamente transcrições de vídeos YouTube
- Permite configurar local de salvamento dentro do vault
- Puxar metadados: título, URL, data de clipping, tags

Configuração em `/settings`:
- Definir nome do vault
- Template padrão de propriedades (source title, URL, date)
- Local padrão: `/raw` para contenúdo não processado

## Visualização

- Graph view: mostra conexões entre todas as notas (evolui com o tempo)
- Backlinks: ver todas as notas que lindam para a atual
- Search: buscar dentro da wiki

## Integração com segundo cérebro

Usado como camada visual e de organização. Funciona com:
- Codex para processamento
- GitHub para backup
