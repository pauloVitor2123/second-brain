## O que é

Formato de texto simples legível e estruturado. Usa símbolos como `#`, `**`, `[]()` para formatar.

## Uso no segundo cérebro

Todo o sistema é baseado em markdown:
- Notas salvam como `.md`
- Fácil de ler e editar manualmente
- Compatível com versioning (GitHub)
- LLMs conseguem processar facilmente
- Obsidian renderiza markdown para visualização

## Vantagens

- Durável (não prende em ferramentas proprietárias)
- Legível mesmo em texto puro
- Suporta links internos `[[nota]]`
- Simples mas poderoso
